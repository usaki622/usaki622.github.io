1.Bootstrap弹框

这个案例可能会用到弹框，我们就使用Bootstrap的弹框吧。先让我们回顾一下Bootstrap是什么：

Bootstrap 是一个流行的开源前端框架，它由 Twitter 开发并开源，用于简化 web 开发的样式和交互设计。以下是 Bootstrap 的一些关键特点：

  1. 响应式设计：
    • Bootstrap 提供了响应式的栅格系统，可以轻松地创建适应不同屏幕尺寸的网站和应用。
  2. 预定义样式和组件：
    • Bootstrap 包含大量的预定义样式和组件，如按钮、表单、导航栏、轮播图等，可以通过简单的 HTML 和 CSS 类来使用，减少了开发者的工作量。
  3. 基于 Less 和 Sass 的样式表：
    • Bootstrap 的样式表是基于 Less 或 Sass 构建的，允许开发者通过变量、混合（mixin）等功能自定义样式。
  4. JavaScript 插件：
    • Bootstrap 提供了一系列的 JavaScript 插件，用于增强页面的交互性，例如模态框、弹出框、滚动监听等。
  5. 跨浏览器兼容性：
    • Bootstrap 经过广泛测试，确保在主流浏览器中具有良好的兼容性。
  6. 社区支持和文档丰富：
    • 由于 Bootstrap 的流行，有庞大的开发者社区，提供了大量的文档、教程和第三方资源，方便开发者学习和使用。
  7. 定制性：
    • Bootstrap 允许开发者通过定制工具选择自己所需的组件和样式，从而减小项目的体积。

使用 Bootstrap 可以加速 web 开发过程，使得页面的样式和交互设计更加一致和专业。开发者可以选择整体引入 Bootstrap，也可以根据需要选择性地使用其中的部分组件和样式。

而这个弹框是Bootstrap提供的一个组件，叫模态框（Modal）。

Bootstrap 模态框（Modal）是 Bootstrap 框架提供的一种交互式组件，用于在页面上显示一块浮动的对话框。模态框常用于显示额外的内容、表单、或者其他需要用户关注的信息，同时阻止用户与页面的其余部分进行交互，直到模态框被关闭。
